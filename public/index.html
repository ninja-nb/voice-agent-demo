Voice AI Agent Sample

Record audio, transcribe, search, get AI answer, and hear TTS output.

Page Summary

This page demonstrates an end-to-end voice AI workflow: audio input -> speech-to-text -> search grounding -> LLM answer -> text-to-speech playback. It supports both single-response mode and live streaming mode.

How to Use

  1. Select an available LLM Provider and Model.
  2. Provide audio by either recording (`Start Recording`/`Stop Recording`) or uploading a file.
  3. Click Run Once for one final response, or Run Live for stage-by-stage updates.
  4. Review transcript, answer, and search evidence in the output panel.
  5. Use the Operations & Debug section at the bottom to monitor status, timing, and runtime events.
Run Once: best for simple testing; one final response. Run Live: shows progress (`transcribing`, `searching`, `answering`) while processing.
